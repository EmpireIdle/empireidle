# Empire Idle - Text World

A browser-based empire-building idle game where you manage resources, build houses, and explore the world.

## Features
- Harvesting and resource management.
- Admin Portal to configure game settings.
- Multiple units and buildings with customizable stats.

## Setup

Clone this repository to get started.

```bash
git clone https://github.com/yourusername/EmpireIdle.git
cd EmpireIdle
open index.html
